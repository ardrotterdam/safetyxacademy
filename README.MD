5. Commit met bericht: `"SEO Update: Internal linking + breadcrumbs + keyword optimization"`

### **Optie 2: Via Git Command Line**

```bash

cd /pad/naar/safetyxacademy

# 2. Kopieer de nieuwe bestanden

git diff

# 6. Push naar GitHub

```
